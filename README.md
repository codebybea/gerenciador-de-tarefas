# Tasks

Projeto desenvolvido para aprender e praticar conceitos de **React**.

## Objetivo

Aplicar na prática os conhecimentos adquiridos sobre React, com foco em:

- Componentes e Props
- Estado e Hooks
- Renderização condicional
- Listas e chaves

## Tecnologias Utilizadas

- React
- JavaScript ES6+
- CSS3

## Como Executar

```bash
npm install
npm start
```
